# Representação em Lista

A lista é a forma mais simples de organizar dados.
Os elementos ficam em sequência.

```txt
[ Ana, Bruno, Carla, Diego, Elisa ]
```

### Teoria aplicada

* Estrutura linear
* Ordem sequencial
* Fácil de armazenar e percorrer
* Cada elemento ocupa uma posição

Exemplo:

| Posição | Aluno |
| ------- | ----- |
| 1       | Ana   |
| 2       | Bruno |
| 3       | Carla |
| 4       | Diego |
| 5       | Elisa |
